# Components Directory

Bootstrap Plus includes a wide variety of UI elements and layouts. Below is an overview of the components available in this guide, organized by their respective sections in the gallery.

## Utilities & Setup
- **CDN Setup**: Core template setup for HTML headers. Include Bootstrap 5 CSS, JS bundle, and Icons.
- **Colors & Background**: Bootstrap utility classes for custom foreground text colors and background colors.
- **Margin & Padding**: Spacing utilities demonstrating outer margin spacing and internal text padding grids.

## Navigation Elements
- **Breadcrumbs**: Display directory path location of the current page in a hierarchical structure.
- **Header Top Section**: An info ribbon bar placed above navigation headers. Displays contact links, opening hours, or dynamic notices.
- **Navigation (Navbar)**: Responsive collapsing navigation layouts featuring menus, dropdowns, and search controls.

## Sliders & Carousels
- **Bootstrap Carousel**: Standard responsive slides slider component featuring built-in overlay captions and control buttons.
- **Owl Carousel Style**: A multi-item sliding deck layout mimicking jQuery Owl Carousel. Displays multiple cards side-by-side with dot pagination.

## Common Components
- **List of Buttons**: Various button styles, states, and outlines.
- **Education Timeline**: Vertical timeline layout for chronological events, perfect for portfolios or resumes.
- **Many Type of Lists**: Unstyled, numbered, icon-based, and description lists.

## Cards & Layouts
- **Card Variations**: Multiple card layouts with images, headers, footers, and stretched links.
- **Multi-Column Grids**: Responsive grid system layouts for displaying content across varied screen sizes.
- **Sidebar Component**: Off-canvas or collapsible sidebar navigation menus.
- **Big Footer**: Comprehensive footer layout with multiple link columns and a newsletter signup form.
- **Simple Footer**: Minimalist footer for smaller projects or landing pages.

## Forms & User Interactivity
- **Post Tags & Comments**: Blog layout elements for tags and discussion sections.
- **Login Form**: Standard user authentication layout with a sleek card design.
- **Feedback Section**: Interactive rating stars and user feedback forms.
- **Real-time JS Function**: Interactive live clock, system uptime, and character counter widget.
