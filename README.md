# Bootstrap Plus - Ultimate Component Guide

Bootstrap Plus is a comprehensive gallery and guide for modern, responsive Bootstrap 5 components. It provides ready-to-use, copy-pasteable HTML snippets tailored for both newcomers and expert developers.

## 🚀 Features

- **Light & Dark Mode**: Integrated theme toggler seamlessly switches between day and night aesthetics.
- **One-Click Copy**: Easily copy HTML snippets directly to your clipboard.
- **Live Search**: Quickly find components, utilities, and widgets using the responsive search bar.
- **Responsive Layout**: A collapsible sidebar and mobile-friendly grid system.
- **Custom Widgets**: Features advanced custom components like Timelines, Owl Carousel style sliders, and real-time JavaScript widgets.

## 🛠️ How to Use

Bootstrap Plus runs entirely client-side. There is no build step, node package, or backend required to use the gallery itself.

1. **Clone or Download** the repository to your local machine.
2. **Open `index.html`** in your preferred web browser.
3. **Navigate** using the left sidebar or use the top search bar to find specific elements (e.g., "Nav", "Card", "Table").
4. **Copy Snippets**: Click the "Copy" button on the top right of any code block to copy the HTML snippet to your clipboard.
5. **Implement**: Paste the copied HTML into your own project. Ensure your project includes the Bootstrap 5 and Bootstrap Icons CDNs in the `<head>` section (as demonstrated in the "CDN Setup" section of the site).

## 📂 Project Structure

- `index.html`: The main gallery page containing all component previews and code blocks.
- `style.css`: Custom CSS providing styling, theme variables, and visual enhancements beyond default Bootstrap.
- `script.js`: Handles interactivity, including theme switching, search indexing, sidebar toggling, code copying, and custom widgets.

## 🔗 Dependencies

The project relies on external CDNs for optimal performance:
- [Bootstrap 5.3.3](https://getbootstrap.com/) (CSS & JS bundle)
- [Bootstrap Icons 1.11.3](https://icons.getbootstrap.com/)
- [Google Fonts](https://fonts.google.com/) (Outfit, Inter, Fira Code)
